# Guide de Test - Application Notaire

## Informations de Connexion
- **URL de l'application**: http://localhost:3001
- **Serveur**: Démarré et fonctionnel ✅

## Checklist de Test

### 1. Page d'Accueil
- [ ] Ouvrir http://localhost:3001
- [ ] Vérifier que la page d'accueil s'affiche correctement
- [ ] Vérifier le titre "Cabinet Notaire Keur Jaraaf"
- [ ] Cliquer sur "Accéder au Tableau de Bord"

### 2. Tableau de Bord
- [ ] Vérifier que le dashboard s'affiche
- [ ] Vérifier la présence de la sidebar avec tous les menus
- [ ] Vérifier les statistiques affichées

### 3. Module Clients
- [ ] Cliquer sur "Clients" dans la sidebar
- [ ] Vérifier la liste des clients
- [ ] Tester le bouton "Nouveau Client"
- [ ] Vérifier les fonctionnalités de recherche et filtrage

### 4. Module Actes
- [ ] Cliquer sur "Actes" dans la sidebar
- [ ] Vérifier la liste des actes
- [ ] Tester le bouton "Nouvel Acte"
- [ ] Vérifier les différents types d'actes disponibles

### 5. Module Templates
- [ ] Cliquer sur "Templates" dans la sidebar
- [ ] Vérifier la liste des templates
- [ ] Tester la création d'un nouveau template
- [ ] Vérifier l'éditeur de templates

### 6. Module Comptabilité (OHADA)
- [ ] Cliquer sur "Comptabilité" dans la sidebar
- [ ] Vérifier l'affichage du plan comptable OHADA
- [ ] Tester les écritures comptables
- [ ] Vérifier le journal, grand livre et balance

### 7. Module Rapports
- [ ] Cliquer sur "Rapports" dans la sidebar
- [ ] Vérifier les statistiques
- [ ] Tester l'export PDF/Excel

### 8. Module Archives
- [ ] Cliquer sur "Archives" dans la sidebar
- [ ] Vérifier le système de classification
- [ ] Tester l'upload de documents

### 9. Module Formalités
- [ ] Cliquer sur "Formalités" dans la sidebar
- [ ] Vérifier les formalités disponibles
- [ ] Tester la création d'une nouvelle formalité

### 10. Module Facturation
- [ ] Cliquer sur "Facturation" dans la sidebar
- [ ] Vérifier la liste des factures
- [ ] Tester la création d'une facture

### 11. Module Rendez-vous
- [ ] Cliquer sur "Rendez-vous" dans la sidebar
- [ ] Vérifier le calendrier
- [ ] Tester la création d'un rendez-vous

### 12. Module Recherche Juridique (IA)
- [ ] Cliquer sur "Recherche Juridique" dans la sidebar
- [ ] Tester la recherche avec l'IA
- [ ] Vérifier les résultats

### 13. Module Administration
- [ ] Cliquer sur "Administration" dans la sidebar
- [ ] Vérifier les paramètres
- [ ] Tester la gestion des utilisateurs
- [ ] Vérifier la gestion des privilèges
